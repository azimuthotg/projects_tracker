### 1. เพิ่ม `/health/` Endpoint สำหรับ NMS Agent Monitoring

- **เหตุผล:** ต้องการให้ระบบ NMS (Network Monitoring System) ตรวจสุขภาพของแอปพลิเคชันได้อัตโนมัติ ทั้งสถานะเว็บและการเชื่อมต่อฐานข้อมูล
- **การดำเนินการ:**
  - เพิ่ม inline view `health()` ที่ root level ใน `config/urls.py`
  - เช็กการเชื่อมต่อ DB ด้วย `connection.ensure_connection()` + `SELECT 1`
  - คืนค่า JSON `{status, db, db_ms}` — วัดเวลา query เป็น ms
  - HTTP **200** เมื่อปกติ / HTTP **503** เมื่อ DB มีปัญหา (degraded)
  - เป็น public endpoint (ไม่ต้อง auth) ที่ `path('health/', ...)`
- **ไฟล์:** `config/urls.py`
- **Commit:** `248709a`
